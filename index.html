4. COLORES, IMAGENES Y VIDEOS EN HTML

Colores

Los colores se representan con un # seguido de 6 letras o numeros.



Imagenes

Para insertar una imagen:


Para editar las dimenesiones de la imagen basta con cambiar los atributos width (ancho) y height (altura).
El tamaño de la imagen se puede ser especificado en pixeles o porcentaje, en caso de no especificarlo se utilizara pixeles.
En el atributo src se escribe el nombre de la foto seguido de punto y de su formato.
En el atributo alt se escribe una descripcion.



Videos

Para incluir un video a la pagina se utiliza la etiqueta video.

- Atributos

src = ponerle nombre
width = anchura
height = altura

controls = permite activar la barra de control del navegador web
autoplay = inicia el video automaticamente una vez cargue la web
muted = para silenciar el sonido del video
loop = para poner el video en bucle
preload = como carga el video. 1.auto : carga completamente, 2. none : no hay carga previa.
poster = se incluye una imagen mientras carga el video



Mapas de Imagenes

Para crear un mapa de una imagen hay que añadir a la imagen el atributo usemap="#nombre".
Creas una etiqueta map con un name para poner el nombre.


En el atributo shape se escribe la forma que puede ser : circle, poly, default y rect.
En las coordenadas dependiendo la forma que tengar se escribiran de distinta forma :

- Circulo : x-centro, y-centro, radio
- Rectangulo : x-izda, y-arriba, x-dcha, y-abajo

6. LISTAS

Elementos de la lista : <li>
Lista desordenada : <ul>
Lista ordenada : <ol>


Lista ordenada



El atributo start permite indicar donde empieza la lista.



El atreibuto type permite ordenar la lista dependiendo de parametros :

- Listas decimales : 1
- Listas alfabéticas en minúsculas : a
- Listas alfabéticas en mayúsculas : A
- Listas de números romanos en minúsculas : i
- Listas de números romanos en mayúsculas : I



Lista desordenada

Los elementos no tienen ningun tipo de orden, simplemente estan enumerados uno por uno.

9. FORMULARIOS

Sirven para enviar informacion del usuario al servidor
La estructura basica de los formularios es :



Campos de entrada de datos

Los atributos que se utilizan para la entrada de datos son :

- type : indica el tipo de datos que vamos a utilizar sus valores pueden ser : text, password, checkbox, “radio”, “submit”, “reset”, “file”, “hidden”, “image” y “button”.
- id : nombre para identificar de forma unívoca al campo
- size : tamaño del campo, número de caracteres que podríamos poner
- name : nombre del campo que se envía desde el formulario al servido

Campos de texto

- Permite insertar texto en un formulario



Contraseñas

Sirve para que en vez de caracteres aparezcan caracteres de asteriscos para evitar que se vea la contraseña



Checkbox

Es una casilla para marcarla, tiene un atributo para preseleccionar la casilla



Radios

Es similar al checkbox pero solo se puede seleccionar una casilla



Ficheros

El campo de entrada de datos nos tiene que dar la posibilidad de acceder al sistema de fichero del dispositivo para poder seleccionar uno



Campos Ocultos

Sirven para enviar informacion oculta



Imagenes

Sirve para enviar imagenes



Areas de Texto

Sirve para que el usuario inserte gran cantidad de datos y estos puedan estar en diferentes líneas



Atributos

- Row : Indica el número de filas que tiene el texto
- Cols ; Indica el número de columnas que tiene el texto
- Name : Nombre del campo


Combos de Opciones

Sirve para que el usuario pueda elegir un elemento o varios de una lista



- Atributos : Name (nombre del campo), size (numero de opciones que aparecen visibles), multiple (permite poder elegir varias opciones)
Para definir las diferentes opciones se utiliza el elemento option



El atributo label indica el texto que aparecerá y value es el valor del ítem. Además existe el atributo selected para marcar como predefinido



Si la lista de opciones es muy grande se utiliza el elemento optgroup



Botones

Mediante los botones se pueden realizar opciones de enviar formularios
Los botones se hacen mediante el input

Tambien se pueden hacer botones com la etiqueta button el cual su funcionalidad es igual que imput



Botones de Envio

Para que el boton sea de envio en el atributo type tiene que poner submit



Botones de Borrado

Para que el boton sea de borrado el type tiene que ser reset
